Variables: En programacion, son la forma en que se almacenan datos individuales para darles uso mas tarde. Podría decirse que en cada variable se almacena un ingrediente para la receta que vas a ejecutar luego.
Strings: Son bloques de texto, palabras para mostrar tal cual se escriben ,y el nombre bajo el cual se guardan las variables de texto. 
Funciones (argumentos, return): Las funciones son bloques de codigo que realizan operaciones especificas programadas previamente. Se las describe con antelacion para luego llamarlas por su nombre y no tener que escribir todo lo que la funcion realiza en cada ocasion que desea utilizarse. Los argumentos son los datos que la funcion podria requerir para ejecutarse, y el return es un comando que le indica a la funcion que debe terminar y dar un resultado si correspondiese.
Declaraciones if . Las declaraciones son ordenes directas para que la computadora realice una o varias acciones. Los if son declaraciones que sirven para corroborar si una o varias condiciones se cumplen, e indicar que se debe hacer se cumplan o no.
Valores booleanos (true, false) Un valor booleano solo tiene dos posibilidades, sí o no. En java se les llama a estas posibilidades true(verdadero) y false(falso).
